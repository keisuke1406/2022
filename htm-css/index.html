令和４年度物理サルクについて

令和４年度物理サルクは〇〇君からの勉強会をしましょうとの提案から始まりました．

勉強会の内容はその日の担当者の，分野についての紹介や，行っている研究の内容，担当者が最近学習したソフト，アプリ等の紹介を行います。

基本週１回の活動を予定していますが，参加者の予定に合わせて適宜変更しています．
